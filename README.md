🎬 Netflix Clone – Modern Streaming UI

A visually rich and responsive Netflix-inspired web application that replicates the look and feel of a modern streaming platform.

This project focuses on delivering a cinematic UI experience, smooth browsing, and dynamic content sections similar to platforms like Netflix.

🌐 Live Demo
https://luckygole.github.io/netflix-clone/

💡 Project Overview

The Netflix Clone is designed to mimic a real-world streaming platform interface, featuring:

Hero banners
Movie rows / categories
Interactive UI elements

The goal of this project is to demonstrate advanced frontend design skills and UI replication ability.

✨ Features
🎥 Cinematic Hero Section
🎞️ Movie/Content rows (like Netflix UI)
🖱️ Smooth hover effects
🎨 Dark-themed modern UI
📱 Fully responsive design
⚡ Fast and lightweight performance
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript / React (if used)
Styling: CSS / Tailwind
API: Movie API (TMDB or similar if used)
Deployment: GitHub Pages / Vercel
